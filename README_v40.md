# Версия 40

Исправления режима экспорта карты:

- добавлено отдельное прямоугольное картографическое поле вокруг выбранного экстента;
- экстент в экспорте рассчитывается автоматически по выборке с буфером около 100 км / не менее 2°;
- подписи градусной сетки теперь можно отдельно включать/выключать и регулировать их размер;
- подписи градусной сетки размещаются внутри картографического поля по его рамке;
- рабочая область отделена от картографического поля светлой рамкой-плашкой;
- увеличен размер экспортного холста;
- заголовок, контекст, сводка и легенда выводятся как отдельные перетаскиваемые блоки вокруг карты;
- белые плашки подписей АТЕ расширены, чтобы текст лучше помещался внутри;
- версия интерфейса обновлена до v40.
