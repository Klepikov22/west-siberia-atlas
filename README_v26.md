Версия v26

Изменения:
- фильтры АТЕ переведены на одну диапазонную шкалу с двумя ручками для населения, площади и плотности;
- панель верхнеуровневых АТЕ стала перетаскиваемой, без кнопок "Все/Снять"; при смене года все верхнеуровневые АТЕ включаются по умолчанию;

- в расширенной диаграмме убраны лишние горизонтальные/вертикальные ползунки, сохранена связанная подсветка сектора и строки легенды;
- обновлена версия и cache-busting до v26.
