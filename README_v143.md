# v143 — чистый контур АТД-1 без рваных внутренних отрезков

Что изменено:

- Контур АТД-1 больше не строится по точному сравнению отдельных сегментов границ АТЕ-2. Старый способ ошибался на слоях, где соседние полигоны имели разные последовательности вершин, и поэтому часть районных/уездных границ попадала в тёмный контур первого уровня.
- Для каждого года создан отдельный растворённый слой `data/admin_l1_dissolved/admin_l1_dissolved_ГОД.geojson`.
- Эти слои построены через Shapely `unary_union` по `admin_superparent / admin_parent`; внутренние кольца в растворённой маске удалены, чтобы не появлялись белые дырки и ложные внутренние контуры.
- Аналитическая геометрия АТЕ-2 в `data/admin/admin_*.geojson` не заменялась: клики, показатели, площади, население и топология остаются на исходных объектах.
- Добавлена тонкая подложка растворённых АТД-1 под административным слоем для режимов `admin_parent` и `admin_superparent`. Она закрывает мелкие белые зазоры между полигонами, но не участвует в кликах и аналитике.
- В экспорте PNG/SVG также используется растворённый контур АТД-1, поэтому внутренние границы АТЕ-2 не должны превращаться в толстые границы первого уровня.

Диагностика: `docs/v143_l1_dissolve_diagnostics.json`.
