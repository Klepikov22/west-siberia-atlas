# Версия 53

Доработка структуры городского и несельского населения для временных слоёв 1918–1939 гг.

## Главное

- Обновлены слои `1918`, `1923`, `1926`, `1930`, `1939`.
- Города, рабочие посёлки и ПГТ вынесены в отдельные модельные полигоны на основе 3-километровых буферов вокруг координат точечных данных.
- Население этих городских/несельских полигонов вычтено из административных единиц-хостов, чтобы не было двойного счёта.
- Исправлена проблема, когда район с городом мог выглядеть как имеющий 100% городского населения.
- Разделены показатели:
  - `strict_city_pop` — собственно города;
  - `worker_settlement_pop` — рабочие посёлки / ПГТ;
  - `broader_urban_pop` — города + рабочие посёлки / ПГТ;
  - `rural_pop_corrected` — остаток населения после вычитания `broader_urban_pop`.

## Методика

1. Использованы предоставленные точки городов и рабочих посёлков с населением по годам.
2. По статусным полям и методическим запискам разделены собственно города и расширенное городское/несельское население.
3. Для каждой точки построен модельный буфер радиусом около 3 км.
4. Буфер привязан к административной единице-хосту по попаданию координаты в полигон соответствующего года.
5. Население городского/несельского полигона вычтено из хост-АТЕ.
6. В статистике интерфейса `urban_pop` трактуется как расширенное городское/несельское население, а собственно города и рабочие посёлки хранятся отдельно.

## Ограничения

- 3 км — модельный радиус, не историческая городская черта.
- Для рабочих посёлков учтены только пункты, присутствующие в предоставленных данных с координатами, населением и годовой логикой включения.
- Значения по модельным годам 1918, 1923 и 1930 остаются реконструкцией, а не прямой переписной статистикой.
