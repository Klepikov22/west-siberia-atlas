# v125 — двухплоскостной анализ приуроченности границ

## Главное

v125 перерабатывает режим анализа происхождения/приуроченности границ после v124: унаследованность больше не считается физико-географическим классом и не конкурирует с реками, озёрами, береговой линией и водоразделами.

## Плоскость 1. Природные факторы

Считаются доли периметра АТЕ по классам:

- береговая линия;
- реки;
- озёра / водная гладь;
- водоразделы сточных бассейнов;
- водоразделы бессточных областей стока;
- болота / заболоченные массивы — методический слот, расчёт требует отдельного векторного слоя болот;
- не объяснено физико-географическими рубежами.

## Плоскость 2. Преемственность / инфраструктура

Параллельно считаются:

- совпадение с границами слоя 2021 года;
- совпадение с низовой муниципальной / сельсоветской сеткой MO_retrospecular;
- приуроченность к железным дорогам, допуск 1,5 км;
- union 2021/MO;
- диагностический показатель прямолинейных необъяснённых участков.

## Важная оговорка по болотам

На текущем шаге был предоставлен только скриншот болотного слоя, а не геоданные. Поэтому поля `nb_wetland_*` заложены, но не рассчитаны. После загрузки векторного слоя болот расчёт можно повторить без изменения интерфейса.

## Техническая логика

Для 2021/MO/ЖД используется не all-or-nothing попадание линии в буфер. Вдоль каждого сегмента берутся точки с шагом около 5 км, и в метрики входит доля длины сегмента, соответствующая доле точек в допуске. Это уменьшает переоценку совпадений на длинных сегментах.
