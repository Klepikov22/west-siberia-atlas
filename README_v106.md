# v106

Добавлены пояснения в окно «Динамика метрик по годам» для группы «Разброс площадей».

## Что изменено

- Версия обновлена до v106.
- В окно мультивременной аналитики добавлен раскрываемый блок «Что здесь проверяется?».
- Для метрик разброса площадей добавлена динамическая карточка «Пояснение»:
  - что означает выбранная метрика простым языком;
  - как читать рост/снижение линии;
  - какой уровень АТЕ сейчас анализируется;
  - выбран ли весь статистический охват или отдельный верхний контекст;
  - краткий автоматический вывод по гипотезе между первым и последним выбранным годом.
- Добавлены пояснения для CV, Gini, p90/p10, q75/q25, max/min, стандартного отклонения, средней, медианной, минимальной и максимальной площади.
- В боковой панели сохранено методическое предупреждение об очистке ряда: без спорных, двоеданческих, слабоконтрольных, неясных территорий и полигонов < 50 км², с исключениями v104.

## Проверка

- `node --check app.js` проходит.
