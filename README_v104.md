# v104

Изменение касается только панели **«Динамика метрик по годам»** и файла `data/topology/multiyear_metrics_by_year.json`.

## Статистический охват v104

По пользовательскому требованию из мультивременной статистики исключены:

- **1926** — Шадринский, Курганский и Ирбитский округа;
- **1930** — Ачинский округ;
- **2021** — Курганская область.

Исходные картографические слои `data/admin/admin_*.geojson`, `data/topology/topology_*.geojson` и `data/topology/topology_nodes_*.geojson` не менялись. Исключения применены только к сводной динамике.

## Пересчитано

Для указанных годов пересчитаны:

- число объектов АТЕ, число верхних/средних/нижних АТЕ;
- суммарная и средняя площадь;
- суммарное, среднее, городское и сельское население, плотность и доля городского населения;
- длина ЖД и плотность ЖД-сети;
- среднее соседство;
- агрегированные графовые показатели после удаления из графа соответствующих узлов и инцидентных рёбер;
- дополнительные иерархические показатели v100.
