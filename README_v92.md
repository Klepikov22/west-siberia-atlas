# v92 — топологический граф и легенда

## Что изменено

- Топологический граф перенесён из отдельного блока под режимом карты в общий блок переключателей слоёв.
- Рёбра и узлы графа теперь фильтруются по текущему административному слою: если пользователь ограничил верхнеуровневые АТЕ, числовые фильтры или выделил конкретную выборку, граф строится только по оставшимся / выбранным объектам.
- Из топологии исключены:
  - спорные территории;
  - двоеданческие / зоны двойной принадлежности;
  - территории с неясным контролем или принадлежностью;
  - территории с особым реконструкционным статусом;
  - города / горсоветы площадью менее 50 км²;
  - микрофрагменты площадью ≤ 1 км².
- Для 1780 г. граф ограничен нормальными объектами Колыванской области; контекстные территории и микрофрагмент Кузнецкого уезда исключены.
- Для 1805 г. удалены рёбра, проходившие через зоны неясной принадлежности у Берёзовского / Мангазейского контура.
- Для 1918 г. исключены территории, подписанные как переданные/подлежащие передаче в Семипалатинский контур, а также спорный Кузнецко-Щегловский фрагмент.
- Для 1923 г. исключены фрагменты с неясной и спорной принадлежностью между губерниями.
- Узлы перестроены после фильтрации рёбер: точки узлов совпадают с endpoints оставшихся рёбер; изолированные узлы сохраняют прежние координаты.
- Легенда топологических классов получила раздельную колонку счётчиков: между подписью класса и числом оставлен интервал 50 px с центральной точкой, счётчик выводится как `N шт.` и не выделяется жирным.

## Проверки

- `node --check app.js` — без ошибок.
- Все GeoJSON/JSON-файлы проекта читаются после пересборки.
- Проверены контрольные годы 1780, 1805, 1918 и 1923: в рёбрах и узлах нет ссылок на исключённые территории; линии рёбер не пересекают специальные/неясные полигоны в этих годах.
