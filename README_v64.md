# Версия 64

- Исправлена подгонка слоя в экспортной рамке: базовый отступ от фактического административного экстента до границ поля — 10 px, километровые буферы работают как ручная добавка.
- Масштабная линейка в экспорте стала реально перетаскиваемой: она получает собственную кликабельную область без видимой подложки.
- Добавлены компактные стили интерфейса для экранов 1920×1080, чтобы панели и таймслайдер не выглядели как при увеличенном масштабе.
