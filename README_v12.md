# Западная Сибирь — веб-MVP v12

Изменения относительно v11:

- Исправлен dissolve северных объектов: фрагменты объединяются по `unit_id`, поэтому городские объекты вроде `г. Надым` не сливаются с Надымским районом.
- В 1926 году схлопнуты фрагменты `Обдорского` объекта.
- Начиная с 1979 года схлопнуты фрагменты `Надымского` района/округа, но отдельные городские объекты оставлены отдельными.
- Режим «Рука» теперь явно оформлен как базовый инструмент: перемещение карты + одиночный выбор кликом.
- Добавлены кнопки инструментов: Рука / Рамка / Полигон.
- Подписи центров сделаны более заметными и с фильтрацией по месту на экране. Если подпись не помещается или пересекается с другой, она скрывается.
- Если для года нет отдельного слоя центров, подписи строятся из поля `center` административного слоя, если оно заполнено.

Открывать на GitHub Pages лучше с параметром:

`?v=12`
