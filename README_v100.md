# v100

- DOM-узлы графа по `topology_nodes_YYYY.geojson` оставлены как надёжный способ отображения, но их слой опущен ниже интерфейса: панели, верхнеуровневый фильтр и таймслайдер теперь перекрывают узлы.
- Размер и цвет JSON-узлов пересчитываются от выбранной метрики узлов; смена метрики сразу обновляет отображение и легенду.
- Рёбра и легенда реагируют на режим стиля рёбер: «по типу связи» / «единый стиль».
- Экспортная легенда получила дополнительные управляемые пункты: классы рёбер, мостовые рёбра, классы узлов по метрике, общий символ JSON-узлов, узлы при мостовых рёбрах.
- В `multiyear_metrics_by_year.json` добавлены показатели иерархии: среднее число нижних АТЕ на единицу верхнего уровня, среднее число нижних АТЕ на прямой родительский уровень, средняя площадь АТЕ верхнего и среднего уровня.
- Диагностика: `docs/v100_multiyear_hierarchy_metrics.csv`.
