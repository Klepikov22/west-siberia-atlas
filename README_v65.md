# Версия 65

Hotfix режима экспорта и компактного интерфейса.

- Исправлена ошибка экспорта `Cannot read properties of undefined (reading 'zoom')`.
- Масштабирование экспортной карты теперь считается относительно центра внутренней рамки карты, а не всего PNG-листа.
- Это убирает перекос западного/восточного отступа, возникавший при правой колонке легенды и сводки.
- Масштабная линейка получила делегированный drag-обработчик внутри SVG и должна перемещаться как отдельный элемент.
- Добавлен более жёсткий компактный режим для Full HD / 1920×1080: панели, кнопки, таймслайдер и плавающие виджеты уменьшаются без необходимости ставить масштаб вкладки 75%.
