# v105 — разброс площадей АТЕ в динамике метрик

Добавлено в модуль **«Динамика метрик по годам»**:

- новая группа показателей **«Разброс площадей»**;
- показатели для верхнего, среднего и нижнего уровня АТЕ:
  - коэффициент вариации площадей, `CV = σ / mean`;
  - индекс Джини по площадям;
  - отношение `p90 / p10`;
  - отношение `q75 / q25`;
  - отношение `max / min`;
  - стандартное отклонение площадей;
  - средняя, медианная, минимальная и максимальная площадь;
- переключатель **«Национальный / верхний контекст»**:
  - весь статистический охват;
  - отдельные верхнеуровневые контексты / национально-административные системы, если они есть в данном году.

Расчётный охват:

- исключены спорные, двоеданческие, слабоконтрольные, неясные и контекстные территории;
- исключены полигоны меньше **50 км²**;
- сохранены пользовательские статистические исключения v104:
  - 1926 без Шадринского, Курганского и Ирбитского округов;
  - 1930 без Ачинского округа;
  - 2021 без Курганской области.

Диагностика:

- `docs/v105_area_dispersion_contexts.csv`;
