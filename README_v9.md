# Western Siberia Historical Atlas — v9

Изменения относительно v8:

- Сохранён одиночный выбор в режиме «Рука / перемещение»: клик по району/уезду добавляет или убирает объект из выборки.
- Сохранены инструменты выборки прямоугольником и полигоном.
- Выполнено объединение фрагментов северных административных единиц, распавшихся после обрезки по океану.
- До 1926 года объединялись объекты, в названии которых встречаются «Берёзовский» / «Обдорский».
- С 1926 года объединялись объекты, в названии которых встречаются «Тазовский», «Ямальский», «Приуральский».
- Площадь и плотность пересчитаны после объединения геометрий.
- Для демографических полей использована сумма уникальных значений внутри объединяемой группы, чтобы не умножать население на число фрагментов после обрезки.

См. отчёт: `docs/v9_dissolve_summary.csv`.
