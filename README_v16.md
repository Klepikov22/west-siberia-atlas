# Western Siberia Atlas — v16

Изменения:

- Добавлена hover-панель: при наведении на полигон/круг/центр через ~0,5 сек появляется мягкая карточка возле курсора.
- Для точечных центров размер круга и hover используют `center_pop_urban`, нормализованное из `Pop_Urban` / городских полей исходников, с fallback на population.
- Правая панель перестроена: карточка выбранного объекта/центра находится выше общей статистики и диаграмм.
- Добавлены аккуратные HTML-подписи центров поверх карты. Они скрываются при пересечениях, у краёв экрана и на мелком масштабе.
- Сохранены режимы v15: кликабельные центры, ЖД-показатели, диаграммы по верхнему уровню, просмотр структуры атрибутов слоя.
