# v98

- Уменьшен размер DOM-узлов графа примерно в 2 раза относительно v97.
- Легенда больше не копируется слепо из интерактивной панели: она пересобирается по реально включённым слоям.
- В режиме экспорта добавлен блок настройки содержания легенды.
- При открытии экспорта видимость основных слоёв и топологических рёбер/узлов берётся из текущего интерактивного режима.
- В SVG-экспорт добавлена отрисовка рёбер смежности и узлов-центроидов, если они включены в экспортных слоях.
