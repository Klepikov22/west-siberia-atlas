# Версия 42

Дополнительно к v41 улучшен экспорт карт.

Что добавлено:

- отдельная настройка **внешней рабочей рамки** экспорта в пикселях;
- отдельная настройка **физических границ картографического поля** в пикселях;
- сохранена отдельная настройка **географического буфера** от выбранных административных полигонов в километрах;
- режим контекста **«Авто-компактный»** — делает более короткий академичный текст для дипломного макета;
- сохранена перетаскиваемость заголовка, контекста, сводки и легенды;
- превью остаётся без внутренних ползунков и без панорамирования карты, чтобы не ломать сетку и подписи;
- версия проекта обновлена до v42.
