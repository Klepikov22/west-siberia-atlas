Версия v36

Что изменено:
- экспортная карта полностью пересобрана: вместо Leaflet/скриншота используется прямой SVG-рендер по GeoJSON-слоям;
- легенда, общая информация и контекст периода теперь размещаются внутри картографического поля и перетаскиваются мышью;
- окно экспорта увеличено, карта занимает существенно больше места;
- добавлены масштабная линейка и лёгкая градусная сетка;
- у рамок картографического поля добавлены подписи градусов с небольшим отступом от края;
- подключена Natural Earth graticules 10° как GeoJSON-слой;
- сохранены форматы A4 horizontal / A4 vertical / широкий экран и режимы оформления для диплома, статьи и презентации.
