# v68

Исправления после проверки v67:

- Восстановлена нормальная компоновка увеличенного окна круговых диаграмм: компактные Full HD-правила больше не уменьшают SVG и центральные подписи внутри lightbox.
- Свернутая панель «Верхнеуровневые АТЕ» укорочена и закрыта от горизонтального скролла: кнопки «Показать все» / «Отменить все» скрыты, «Развернуть» не выступает за рамку.
- Экспорт карты оптимизирован:
  - добавлены debounce и кеширование SVG-карты, чтобы изменение текста, карточек и соседних контролов не пересобирало тяжелые слои без необходимости;
  - масштабная линейка получила исправленный drag-hitbox и больше не улетает в левый верхний угол;
  - перетаскивание линейки не активирует рамку картографического поля.
- Градусная сетка в экспорте стала динамической:
  - шаг выбирается по охвату;
  - подписи обрезаются clipPath внутренней рамки и не выходят за картографическое поле.
- Добавлен ползунок приглушения окружения вне текущего слоя / выборки:
  - гасит гидрографию и железные дороги за пределами выбранных административных полигонов;
  - диапазон непрозрачности 0–85%.
- Для протяженности ЖД добавлены классовые шкалы с округлением до 10 км:
  - линейная,
  - квантильная,
  - геометрическая.
- Для Full HD повторно ослаблен нижний clamp у перетаскиваемых панелей фильтров и верхнеуровневых АТЕ.

Проверка:

- `node --check app.js` проходит.
