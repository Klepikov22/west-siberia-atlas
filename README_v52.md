# Версия 52

Добавлены новые временные срезы и подключены в таймслайдер:

- 1918 г.;
- 1923 г.;
- 1930 г.

Новые слои приведены к текущей структуре административных слоёв проекта:

- `unit_id`;
- `year`;
- `name`;
- `unit_type`;
- `admin_parent`;
- `admin_intermediate`;
- `admin_superparent`;
- `center`;
- `population`;
- `urban_pop`;
- `rural_pop`;
- `urban_share`;
- `area_km2`;
- `density`;
- `source_layer`;
- технические поля качества расчёта населения.

Население для новых слоёв рассчитано оценочно:

- для 1918 и 1923 гг. — по слоям 1914 и 1926 гг.;
- для 1930 г. — по слоям 1926 и 1939 гг.;
- метод: ареальная интерполяция по площадному пересечению + линейная временная интерполяция между соседними слоями.

Добавлены поля качества:

- `population_method`;
- `population_prev_ref_year`;
- `population_next_ref_year`;
- `population_prev_areal_estimate`;
- `population_next_areal_estimate`;
- `population_prev_coverage`;
- `population_next_coverage`;
- `population_confidence`.

Для 1930 г. добавлена группировка округов по вышестоящему уровню:

- Сибирский край;
- Уральская область.

Также добавлены режимы отображения:

- промежуточный уровень АТД;
- вышестоящая группировка.

Ограничения:

- для 1930 г. в исходном слое отсутствуют названия районов, поэтому временно созданы технические подписи вида `Безымянный район №...` внутри соответствующего округа;
- численность населения является модельной оценкой, а не первичной статистикой.
