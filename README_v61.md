# Версия 61

Точечная правка интерфейса экспорта и панели фильтров.

Изменения:
- восстановлено поведение редактирования заголовка в экспорте по логике v59;
- внутренняя рамка карты теперь имеет два состояния:
  - обычное: рамку можно двигать;
  - редактирование: включается одиночным кликом по рамке, появляются направляющие и ручки растягивания, движение рамки блокируется;
- после растягивания внутренняя рамка остаётся в режиме редактирования до клика вне поля рамки;
- клик вне рамки карты возвращает обычный вид и обычное перемещение;
- для панели фильтров включена принудительная вертикальная прокрутка;
- у нижнего подблока фильтра задан фиксированный нижний отступ 15 px.
