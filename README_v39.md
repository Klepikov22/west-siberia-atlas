# Версия v39

- Исправлено поведение pan/zoom в режиме экспорта: SVG больше не пересоздаётся на каждом движении, поэтому карта не теряет интерактивность.
- Увеличен минимальный экспортный зум: карта стартует крупнее.
- Масштабная линейка теперь пересчитывается при изменении зума.
- Градусная сетка, подписи сетки, полигоны, реки, круги населения и подписи АТЕ двигаются/масштабируются как единое картографическое тело.
- В экспорт включаются все речные линии без bbox-фильтрации и slice-ограничения, чтобы гидрография не обрезалась.
- Подписи АТЕ одного порядка имеют единый стиль; генерализация влияет только на отбор, а не на размер/жирность.
