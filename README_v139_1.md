# v139.1 hotfix — динамика метрик

Исправлена ошибка открытия панели «Динамика метрик по годам»: браузер падал на `insertBefore`, потому что v120-патч пытался вставить блок управления перед узлом, который не всегда был прямым ребёнком контейнера `.topology-trend-controls-v106/.v91`.

Что изменено:
- `APP_VERSION = 139.1`;
- `index.html` подключает `app.js?v=139.1` и `style.css?v=139.1`;
- `manifest.json` синхронизирован на `v139.1`;
- `insertControls()` теперь вставляет блок через реального родителя anchor-узла;
- healthcheck дополнен регрессионной проверкой против старого небезопасного паттерна.
