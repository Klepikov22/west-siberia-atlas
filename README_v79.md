# v79

Правки населения и именования для 1926 и 1930 годов.

- 1926: сверка районных итогов Западной Сибири с таблицей III переписи 1926 года; точные городские значения вместо округлений; сняты выявленные ошибки Барнаульского/Шадринского, Томского, Бийского, Каменского и ряда городских узлов.
- 1930: точные городские значения на 1/I/1930 по Шнейдеру; проименованы и пересчитаны районы Барнаульского и Каменского округов по справочнику «Округа и районы Сибирского края».
