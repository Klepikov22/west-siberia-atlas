# v130 — продвинутый граф как полноценные слои

Исправления после проверки v129:

- продвинутый граф разделён на два независимых слоя:
  - рёбра потенциальной связности;
  - узлы продвинутого графа / JSON-точки;
- добавлены отдельные чекбоксы управления, аналогично базовому топологическому графу;
- рёбра и узлы больше не завязаны на один общий переключатель;
- продвинутый граф отрисовывается поверх административного слоя и не зависит от режима заливки карты;
- метрики продвинутого графа добавлены в основной селектор режима карты как режимы заливки АТЕ;
- значения `adv_*` присоединены к административным слоям `admin_YYYY.geojson` по `unit_id`;
- легенда теперь отдельно расшифровывает рёбра и узлы продвинутого графа;
- легенда продвинутого графа попадает и в обычную легенду, и в экспортный блок легенды.

Интерпретация: слой остаётся эвристическим графом потенциальной связности, а не реконструкцией фактической транспортной доступности.
