# v144 — 4 краски для стилей защиты ВКР

Что изменено:

- Для стилей `ВШЭ · защита ВКР`, `ВШЭ · защита ВКР · красная` и `ВШЭ · защита ВКР · синяя` добавлена предрассчитанная схема четырёх красок.
- Работает в категориальных административных режимах:
  - `admin_superparent`;
  - `admin_parent`;
  - `admin_intermediate`.
- Цвета назначаются не случайно и не по названию, а по графу смежности растворённых административных групп.
- Соседние единицы верхнего / промежуточного уровня не получают одинаковую заливку.
- Аналитические полигоны АТЕ-2, площади, население, смежность, клики и топологические расчёты не изменялись.
- Для каждого года добавлен файл `data/fourcolor/fourcolor_ГОД.json`.

Диагностика: `docs/v144_fourcolor_diagnostics.json`.
