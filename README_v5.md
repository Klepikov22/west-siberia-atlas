# Western Siberia Historical Atlas — v5

Изменения версии v5:

- убран выпадающий выбор года; вместо него — компактная временная шкала с точками;
- интерфейс переведён в плавающие панели поверх карты, без глобальных полос прокрутки страницы;
- отключены "бегающие" sticky-tooltip у полигонов; подписи теперь статические, в центре объектов, и скрываются у краёв экрана;
- круги населения снова вынесены выше административного слоя, кликабельны и имеют подсказку с населением;
- легенда дополнена градацией кругов населения;
- гидрография разнесена по порядку: реки ниже озёр/океана, озёра и океан перекрывают реки;
- фильтр водохранилищ: до 1959 года объекты Reservoir / Водохранилище / Vodokhranilishche не показываются;
- добавлен cache-busting `?v=5`, чтобы GitHub Pages и браузер не подмешивали старые GeoJSON и старые JS/CSS;
- экстент расширен примерно на 200 км вокруг текущих границ данных.

Файлы для публикации на GitHub Pages нужно загружать целиком: `index.html`, `style.css`, `app.js`, папку `data/`, папку `docs/`.
