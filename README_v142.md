# v142 — белая подложка экспорта и отдельный красный стиль ВШЭ

Что исправлено:

- В окне экспорта для подложек `Почти белая`, `ВШЭ · защита ВКР`, `ВШЭ · защита ВКР · красная` и `ВШЭ · защита ВКР · синяя` теперь принудительно используется чистый белый фон `#ffffff`.
- Исправлен старый жёстко прошитый цвет экспортного SVG `#eef3ef`, из-за которого вокруг внутренней рамки карты оставалась зеленовато-серая подложка.
- Исправлен фон `html2canvas` при сохранении PNG: для белых стилей он теперь тоже `#ffffff`, а не `#f7f5ef`.
- Добавлен отдельный пункт стиля `ВШЭ · защита ВКР · красная` (`hseDefenseRed`). Визуально он наследует красную ВШЭ-схему v140, но теперь вынесен как отдельный вариант рядом с синей схемой.
- Для главной карты у `Почти белая` и ВШЭ-стилей отключены декоративные градиенты подложки.
