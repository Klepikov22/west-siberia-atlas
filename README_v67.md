# v67

Исправления по экспорту, Full HD-интерфейсу и легендам:

- экспортная автоподгонка теперь заново центрирует слой/выборку по проецированному экстенту и вписывает объект в рамку с базовым зазором 10 px;
- добавлен независимый ползунок масштаба карты внутри внутренней рамки: ±1,5 zoom-step от автоохвата, без растягивания самой рамки;
- экспортный SVG обновляется с защитой от гонок асинхронной отрисовки, а изменение масштаба применяет transform без полной пересборки превью;
- масштабная линейка получила HTML-hitbox поверх внутренней рамки: при наведении она имеет приоритет над перемещением/растягиванием рамки;
- для Full HD снят преждевременный нижний барьер у перетаскиваемых блоков фильтров и верхнеуровневых АТЕ;
- свернутая панель верхнеуровневых АТЕ стала короче: кнопки “Показать все” и “Отменить все” скрыты, остается только разворачивание;
- в развернутой панели кнопки выбора верхнеуровневых АТЕ выровнены в одну левую строку;
- круговые диаграммы в правой панели сжаты и выровнены для режима 1920×1080;
- для плотности населения, плотности железных дорог и доли городского населения добавлены фиксированные классы;
- добавлен переключатель шкалы заливки: непрерывная, фиксированные классы, квантили с округлением, геометрическая шкала с округлением.

Проверка: `node --check app.js`.
