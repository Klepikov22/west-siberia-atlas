# Версия v82

## Изменения

1. Добавлены новые ранние административные слои:
   - 1798
   - 1805
   - 1809
   - 1821
   - 1838

2. Новые слои приведены к общей структуре атрибутов проекта:
   - `unit_id`
   - `year`
   - `name`
   - `unit_type`
   - `admin_parent`
   - `admin_intermediate`
   - `admin_superparent`
   - `center`
   - `population`
   - `urban_pop`
   - `rural_pop`
   - `urban_share`
   - `area_km2`
   - `density`
   - `rail_length_km`
   - `rail_density_km_1000`
   - `rail_segments_count`
   - `data_confidence`
   - `atd_hierarchy`

3. Для ранних слоёв население в большинстве исходников не задано, поэтому поля населения оставлены пустыми. Площадь рассчитана из `Shape_Area`, ЖД-показатели для этих лет выставлены нулевыми.

4. В новых слоях объединены одноимённые фрагменты исходной геометрии, чтобы не было дублей по `name + admin_parent + unit_type`.

5. Слои добавлены в `manifest.json` и на таймслайдер.

6. В таймслайдер добавлен фильтр `XVIII век`, чтобы слой 1798 можно было выбирать не только через “Все годы” / “Имперский период”.

7. Для 2021 года изменена вышестоящая группировка:
   - `Алтайский край` остаётся отдельной группой;
   - `Республика Алтай` остаётся отдельной группой;
   - объединение Республики Алтай в Алтайский край для 2021 года отключено.
