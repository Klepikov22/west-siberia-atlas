# v91 — правка топологического графа и окна динамики

## Что изменено

- Исправлен показ рёбер и узлов смежности при включении чекбокса **«Показать рёбра и узлы смежности»**:
  - граф теперь рендерится отдельным устойчивым обработчиком v91;
  - рёбра и узлы подняты в отдельные Leaflet panes с высоким z-index;
  - добавлена защита от асинхронных гонок при смене года / режима / чекбокса.

- Перестроены `data/topology/topology_nodes_*.geojson` для всех лет:
  - координаты узлов с рёбрами теперь берутся из фактических endpoints рёбер соответствующего года;
  - изолированные узлы без рёбер сохранены на прежней точке;
  - узлы должны лежать ровно в точках схождения рёбер.

- Очищена легенда топологии:
  - убрано дублирование формулировки «Топология: …» в заголовке классов;
  - счётчики классов и рёбер сделаны не жирными и менее навязчивыми;
  - блоки разделены на «Классы узлов» и «Рёбра и узлы смежности».

- Переработано окно **«Динамика метрик по годам»**:
  - заголовки параметров приведены к верхнему регистру через CSS;
  - добавлен выбор шкалы: линейная / логарифмическая log10;
  - добавлен выбор цвета линии и точек;
  - добавлены подписи значений над точками и настройка размера подписи.

## Проверка

- `node --check app.js` проходит без ошибок.
- Проверена согласованность координат: endpoints всех рёбер совпадают с координатами соответствующих узлов в `topology_nodes_*.geojson`.
