# Western Siberia Historical Atlas — v10

Изменения относительно v9:

- Исправлена демографическая нормализация слоя 1939 года: `urban_pop` теперь выводится из `population - rural_pop`, а `urban_share` пересчитывается для всех объектов 1939 года.
- Добавлена кнопка «Структура атрибутов слоя»: показывает список полей текущего административного слоя, типы значений, заполненность и примеры.
- Административные подписи временно отключены.
- Подписи теперь относятся к центрам.
- Центры отображаются пропорциональными кругами по полю `population` и имеют tooltip/popup.
- В легенду добавлена секция по центрам.

Загрузка на GitHub Pages: заменить всё содержимое репозитория, кроме скрытой папки `.git`, на содержимое этой папки, затем `Commit to main` и `Push origin`.
