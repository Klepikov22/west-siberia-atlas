# Версия v88

## Топологический граф и аналитика связности

1. Граф смежности теперь включается строго чекбоксом «Показать рёбра и узлы смежности». Топологический режим заливки карты сам по себе больше не включает рёбра и узлы.
2. Узлы графа вынесены в отдельную Leaflet-pane поверх рёбер и полигонов, чтобы они не пропадали за заливкой АТЕ.
3. Рёбра графа получили типологию:
   - внутри одного текущего родителя;
   - между единицами одной вышестоящей группы;
   - межродительские / межобластные / межгубернские связи.
4. Легенда теперь показывает числовые диапазоны выбранной топологической метрики и количество рёбер по типам связей.
5. Добавлено модальное окно «Динамика метрик по годам» с линейным графиком и выбором годов.
6. Добавлен файл `data/topology/topology_metrics_by_year.json` и CSV-сводка `docs/v88_topology_metrics_by_year.csv`.
