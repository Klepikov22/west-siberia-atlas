# v94 — устойчивость границ по настоящим общим границам АТД

Главное исправление v94: слой «Устойчивые рубежи АТД» больше не строится по связям между центрами/узлами графа. Геометрия слоя — это настоящие сегменты общих границ полигонов АТЕ.

## Методика

1. Берутся административные полигоны всех годовых срезов из `data/admin`.
2. Исключаются специальные и недостоверные объекты: спорные, двоеданческие, слабоконтрольные, неясные, контекстные, передаваемые территории, микрофрагменты и малые города/горсоветы.
3. Внутри каждого года извлекаются реальные общие границы между оставшимися полигонами АТЕ.
4. Для каждого настоящего сегмента границы считается геометрическая близость к настоящим сегментам границ других лет.
5. Допуск:
   - 15 км, если середина сегмента севернее/на 59° с. ш.;
   - 10 км, если южнее 59° с. ш.
6. На карту выводится исходный реальный сегмент границы референсного года с атрибутами устойчивости.

## Что изменено

- Новый расчётный слой: `data/stability/boundary_stability_v94.geojson`.
- Слой подключён через `data/manifest.json`.
- Tooltip теперь прямо показывает, что используется расстояние «линия-к-линии».
- Убрана ошибочная логика v93, где фактически получалась паутина связей по центральным точкам.

- Исходных настоящих сегментов общих границ: 14 803.
- Выходных сегментов устойчивости: 13 840.
- Временных срезов: 30, включая 2021.
