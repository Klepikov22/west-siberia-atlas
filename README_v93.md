# v93

Исправления и добавления:

- Пересобран показ топологического графа: отображение рёбер и узлов теперь использует данные узлов/рёбер v92, но выводится более надёжным механизмом поверх карты с яркой обводкой и halo-подложкой, чтобы граф не терялся на административной заливке.
- Граф фильтруется по текущему административному слою и текущей выборке: если выбраны АТЕ, остаются только узлы выбранных АТЕ и рёбра между ними.
- В панель слоёв добавлены отдельные переключатели **Рёбра** и **Узлы** внутри единого блока «Топологический граф».
- Панель «Динамика метрик по годам» расширена с топологии до общего набора показателей:
  - число АТЕ верхнего, среднего и нижнего уровня;
  - суммарная и средняя площадь;
  - население, плотность, городское/несельское население и его доля;
  - суммарная длина ЖД, плотность ЖД-сети и сумма ЖД-сегментов по АТЕ;
  - среднее соседство;
  - прежние графовые и топологические метрики.
- Добавлен предрасчёт `data/topology/multiyear_metrics_by_year.json` и ссылка на него в `data/manifest.json`.
- Версия обновлена до `APP_VERSION = 93`, `app.js?v=93`, `style.css?v=93`.

Проверка: `node --check app.js`.
