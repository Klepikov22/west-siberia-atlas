# v93 — устойчивость границ АТД

Добавлен аналитический слой геометрической устойчивости границ Западной Сибири.

## Методика

Источник — очищенные рёбра смежности АТЕ из топологического блока v92, построенные по основным административным полигонам. В исходные рёбра не попадают спорные территории, зоны слабого/неясного контроля, двоеданческие территории, специальные статусы, передаваемые территории и малые городские полигоны.

Для каждого исторического ребра выполнено разбиение на отрезки примерно по 25 км. Для середины каждого отрезка ищутся близкие отрезки других годовых срезов:

- севернее или на 59° с. ш. — допуск 15 км;
- южнее 59° с. ш. — допуск 10 км.

Метрика является геометрической: она показывает повторяемость близкого прохождения границы, а не тождество административного смысла рубежа.

## Интерфейс

В левую панель добавлен блок «Устойчивость границ»:

- включение слоя «Устойчивые рубежи АТД»;
- минимальное число временных срезов;
- стиль линий: класс/монохром/по среднему смещению;
- подписи наиболее устойчивых рубежей;
- размер подписи.

## Данные

- `data/stability/boundary_stability_v93.geojson` — линейный слой устойчивости;
- `data/stability/boundary_stability_v93_summary.json` — сводка расчёта;
- `docs/v93_boundary_stability_summary.csv` — краткая сводка;
- `docs/v93_boundary_stability_by_count.csv` — распределение по числу совпадающих срезов.
