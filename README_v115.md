Правки слоя 1947 г.:
- Горно-Шорский район переименован в Мысковский район.
- Старый наложенный полигон Мыскинского горсовета заменён компактным 3-км буфером по центроиду внутри Мысковского района; население 15 779 чел. сохранено в новом городском контуре.
- Колпашевский горсовет / район переименован в Колпашевский район, геометрия обрезана по Чаинскому району; Парабельский и Верхне-Кетский районы вырезаны по его финальной маске.
- Для Колпашевского района задано 18 500 чел. городского населения, остальное — сельское.
- Проверены крупные городские контуры 1947 г.; явно заниженные значения относительно внутренних контрольных рядов 1939/1959 исправлены геометрической интерполяцией. Бийск: 47 571 → 102 109 чел.
- Пересчитаны площадь, плотность, соседство, topology_1947.geojson, topology_nodes_1947.geojson, строка 1947 в topology_metrics_by_year.json и multiyear_metrics_by_year.json.
