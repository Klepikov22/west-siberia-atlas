# Western Siberia Atlas — web MVP v2

Изменения относительно первой версии:

- Добавлен светлый стиль интерфейса по умолчанию.
- Добавлен переключатель темы: светлая / тёмная.
- Настроены более плавные переходы масштаба Leaflet:
  - zoomSnap: 0.25
  - zoomDelta: 0.5
  - wheelPxPerZoomLevel: 110
  - flyToBounds для обзорного вида
- Добавлена мягкая «резиновая» рамка экстента:
  - maxBounds
  - maxBoundsViscosity
  - возврат внутрь рабочей области после выхода за пределы.
- Рельеф отключён по умолчанию.
- Bounds рельефа временно приведены к экстенту административных слоёв, чтобы он не улетал отдельно от карты.

Как обновить GitHub Pages:

1. Распаковать архив.
2. В репозитории GitHub открыть Add file → Upload files.
3. Загрузить содержимое этой папки с заменой старых файлов.
4. Commit changes.
5. Подождать 1–3 минуты на обновление Pages.

Старая версия не исчезает полностью: GitHub сохраняет её в истории коммитов. Но публичная ссылка будет показывать последнюю версию из ветки main.
