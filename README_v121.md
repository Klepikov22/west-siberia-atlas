# v121

Добавлен режим карты «Возраст районов».

## Методика

Возраст районов — это прокси-показатель для визуальной картины устойчивости районной сетки. Он рассчитывается для районоподобных единиц с 1926 г. и далее:

- базовый год — 1926;
- наследование определяется по совпадению / близкому написанию названия и смещению центроида;
- при смене названия используется консервативная проверка геометрической преемственности;
- городские единицы, горсоветы, городские округа и центры округов не участвуют;
- показатель не является юридической датой учреждения района.

## Новые поля в GeoJSON

- `district_age_years`
- `district_age_class_10`
- `district_age_basis`
- `district_age_confidence`
- `district_age_predecessor`
- `district_age_predecessor_year`
- `district_age_distance_km`
- `district_age_name_similarity`
- `district_age_overlap_iou`
- `district_age_note`

## Интерфейс

- новый режим карты: «Возраст районов»;
- легенда с классами через 10 лет: 0–9, 10–19, …, 90 лет и старше;
- карточка объекта показывает возраст, год происхождения и предшественника;
- в таблицы метрик добавлены средний, медианный и максимальный возраст районов.
