# v75

Проверка и чистка дублей в поздних слоях АТД.

## Исправлено

- Проверены слои `1970`, `1979`, `1989`, `2021`.
- Объединены технические фрагменты с повторяющимся `unit_id`: статистические поля населения сохранены один раз, площадь и ЖД-метрики суммированы по геометрическим фрагментам.
- В `1989` агрегированы одноимённые городские и районные компоненты в одну строку АТЕ с разнесением населения по `urban_pop` / `rural_pop` и пересчётом `urban_share`.
- В `2021` удалены пустые безымянные слои-сливеры без населения и объединены остаточные одноимённые фрагменты.
