Версия v32

Что добавлено:
- настройки символов населения: круги или столбцы;
- ползунки минимального и максимального размера символов населения;
- методы нормирования: квадратный корень, линейное, логарифмическое, квантильное;
- новый стиль «Матча латте» для полигонов и подложки;
- адаптация цветовой шкалы количественных режимов под стиль полигонов;
- легенда теперь показывает текущий тип символов и текущий метод нормирования.
