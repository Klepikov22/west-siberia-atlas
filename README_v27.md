Версия v27

Что изменено:
- добавлены кнопки сворачивания/разворачивания для плавающих панелей «Фильтры АТЕ» и «Верхнеуровневые АТЕ»;
- скрыта служебная категория с пустым верхнеуровневым АТЕ;
- категория «Прочие» в круговых диаграммах теперь получает нейтральный светло-серый цвет независимо от выбранной палитры;
- увеличены боковые отступы и внутренние поля строк легенды в расширенном окне диаграмм;
- экстент карты расширен на запад и восток примерно на 275 км для захвата соседних территорий подложкой.
