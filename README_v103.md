# Russia Interactive Map v103

Обновление v103:
- исправлен недоучёт населения в панели «Динамика метрик по годам» с 1918/1923/1926 и далее;
- суммарное, среднее, городское/сельское население, доля городского и плотность населения теперь пересчитаны из административных GeoJSON по всем аналитическим объектам слоя, включая малые городские полигоны < 50 км²;
- топологические и соседские метрики по-прежнему не считают малые города, спорные/слабоконтрольные и служебные объекты узлами/соседями.
