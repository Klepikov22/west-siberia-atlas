# v69

Точечный выпуск после v68.

## Исправлено

- В экспортной карте железные дороги больше не обрываются раньше края картографического поля:
  - для отбора ЖД-сегментов используется небольшой служебный контекстный запас вокруг экспортного охвата;
  - сам экстент и пользовательские настройки рамки не меняются;
  - для линейных слоёв добавлен микрозапас SVG-обрезки, чтобы штрихи не срезались по пикселю на границе внутренней рамки.

- Для режима «Длина железных дорог» классы легенды теперь округляются кратно 50 км.

## Оптимизация экспорта

- Экспортное превью перестало пересобирать всю HTML-компоновку при изменениях, которые затрагивают только карту.
- SVG-карта повторно используется по кеш-ключу; положение масштабной линейки и точная прозрачность приглушения применяются поверх уже построенного SVG.
- Уменьшен лимит тяжелого SVG-кеша, чтобы не раздувать память при длительной работе с экспортом.
