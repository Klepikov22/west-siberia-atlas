# v21 — правки 1926 года и крупные диаграммы

## Что изменено

1. Обновлены файлы 1926 года:
   - `data/admin/admin_1926.geojson`
   - `data/centers/centers_1926.geojson`

2. Для административного слоя 1926 года нормализованы поля:
   - `urban_pop`
   - `rural_pop`
   - `urban_share`
   - `population`
   - `density`

   Расчёт городского населения привязан к `centers_1926_final.geojson`; для городов-дублей используется один вклад в суммарную статистику. Сводка записана в `docs/v21_1926_urban_summary.csv`.

3. В интерфейсе добавлено крупное окно просмотра круговых диаграмм:
   - клик по диаграмме в правой панели открывает увеличенную диаграмму поверх карты;
   - окно закрывается кнопкой `×`, кликом по затемнению или клавишей `Esc`;
   - при мультивыборке диаграмма пересчитывается по выбранным объектам;
   - добавлена анимация увеличения карточки с эффектом `liquid glass`.

4. Версия приложения обновлена до `v21`, cache-busting параметры изменены на `?v=21`.

## Проверка

- `node --check app.js` — синтаксис JavaScript корректен.
- Все GeoJSON/JSON-файлы успешно парсятся.
- Для слоя 1926 года сумма населения сходится: `urban_pop + rural_pop = population`.
