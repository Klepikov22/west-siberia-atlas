# v129 — продвинутый граф потенциальной связности

Добавлен отдельный граф связности, не заменяющий базовый топологический граф смежности.

## Источники
- Сибирский тракт: активные участки по `Year_начала_движ` / `Год_закрытия`, только до 1897 г.
- Основные дороги юга Томской губернии: с 1850 г. до 1897 г.
- Судоходные реки и каналы: как водные коридоры до 1876 г.; слабее в 1897–1914 гг.
- Железные дороги: с 1897 г. по `year_open` / `year_close`.
- Высоты более 400 м: сильный орографический барьер.
- Прочие реки: слабый барьер, если рядом нет судоходного коридора.

## Логика
Используется существующий граф смежности `topology_YYYY.geojson`. Для каждого ребра рассчитываются:
- тип коридора / барьера;
- проходимость;
- импеданс;
- сила связи;
- сетевые метрики узлов продвинутого графа.

Рёбра с сильным рельефным барьером без транспортного коридора остаются в линейном слое, но исключаются из сетевых метрик.

## UI
В панели топологии добавлен блок **«Продвинутый граф · коридоры и барьеры»** с включением слоя и выбором метрики узлов.
