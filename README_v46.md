# Версия 46

Пересобранный пакет после проблемы с доступностью архива v45.

- Версия приложения обновлена до v46.
- Сохранена отдельная плавающая кнопка «Экспорт карты» справа по центру экрана.
- У кнопки добавлена явная всплывающая подсказка «Режим экспорта карты» при наведении.
- Сохранён hotfix без рекурсивного вызова `ensureExportFlags()` / `syncExportContextText()`.
