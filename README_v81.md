# Версия v81

## Что изменено

1. Для слоёв после 1939 г. заполнены и нормализованы иерархические поля:
   - `admin_intermediate` = текущее областное / автономное деление;
   - `admin_superparent` = укрупнённая вышестоящая группировка.

2. В `admin_superparent` выполнены объединения:
   - `ХМАО` и `ЯНАО` → `Тюменская область`;
   - `Ханты-Мансийский автономный округ` и `Ямало-Ненецкий автономный округ` → `Тюменская область`;
   - `Горно-Алтайская автономная область` / `Республика Алтай` → `Алтайский край`.

3. Для слоя 1926 г. исправлена иерархия:
   - `Славгородский` нормализован до `Славгородский округ`;
   - для `Славгородского округа` задана вышестоящая группа `Сибирский край`;
   - для `Тобольского округа` задана вышестоящая группа `Уральская область`.

4. Обновлена палитра категориальной окраски:
   - расширены тематические палитры, чтобы соседние округа и области не получали одинаковый цвет только из-за нехватки цветов в наборе.

5. Сброс категориальных цветов теперь делается при смене года и режима показа, чтобы палитра пересобиралась корректно для текущего слоя.
