# Western Siberia Historical Atlas — v14

Изменения относительно v13:

1. Подписи центров переведены на другой механизм — постоянные Leaflet tooltips, привязанные к невидимым маркерам-якорям. Это убирает проблему съезда подписей вниз.
2. В слое 1897 удалён наложенный служебный фрагмент `в составе Томского округа`, который лежал поверх Томского округа и не имел собственной статистики.
3. Для инструментов `Рамка` и `Полигон` добавлено перемещение карты через зажатую среднюю кнопку мыши (СКМ).
4. Остальная логика v13 сохранена: правая статистика, просмотр всех атрибутов объекта, выборка и структура атрибутов слоя.

Открывать после публикации: `?v=14`.
