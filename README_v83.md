# Версия v83

## Главное

1. Добавлены и стандартизированы ранние слои АТД:
   - 1745
   - 1765
   - 1780
   - 1783

2. Слои 1798, 1805, 1809, 1821 и 1838 заменены на повторную стандартизацию из исходников без механического объединения одноимённых фрагментов.

3. Особые / реконструкционные территории больше не dissolve-ятся и не удаляются. Для них добавлены служебные поля:
   - `special_status`
   - `special_status_code`
   - `is_special_reconstruction_area`
   - `hatch_pattern_code`
   - `include_in_analytics`
   - `include_in_selection`
   - `boundary_confidence`

4. Стандартизирован отдельный слой Колывано-Воскресенского горного ведомства:
   - `data/special/kolyvan_special_layer.geojson`
   - в manifest добавлен как `special_overlays.kolyvan_mining_department`
   - в таймслайдер пока не включён как основной административный слой.

5. Сохраняется правка v82: в 2021 году Алтайский край и Республика Алтай в `admin_superparent` остаются отдельными группами.

## Примечание

Отрисовка штриховки особых статусов, исключение из аналитики и отключение интерактивной выборки для таких территорий будут реализованы отдельным этапом, когда будет добавлен полный набор ранних слоёв.
