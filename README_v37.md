Версия v37

Экспортный режим:
- увеличено рабочее окно и картографическое поле;
- добавлена коническая проекция Ламберта (Lambert Conformal Conic) с центральным меридианом 75°E;
- оставлен fallback на экранную/меркаторскую проекцию;
- добавлены отдельные переключатели слоёв: АТЕ, гидрография, Ж/Д, круги населения, подписи, сетка, масштабная линейка;
- добавлены фильтры экспортируемых АТЕ по минимальному населению и минимальной площади;
- добавлена настройка генерализации подписей: не показывать, только крупнейшие, сбалансированно, плотнее;
- легенда, сводка и контекст остаются перетаскиваемыми внутри картографического поля.
