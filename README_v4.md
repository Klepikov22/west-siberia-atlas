# Исторический веб-атлас Западной Сибири · v4

Изменения относительно v3:

1. Убран выпадающий список годов. Вместо него добавлена горизонтальная временная шкала поверх карты.
2. Исправлена причина появления внешних полос прокрутки страницы на широком экране: теперь прокрутка живёт внутри боковых панелей, а не у всей страницы.
3. Зум настроен на центр карты (`scrollWheelZoom: 'center'`), чтобы при приближении/отдалении не было ощущения сдвига вправо-вниз / влево-вверх.
4. Сохранена мягкая рамка экстента, но её влияние сделано спокойнее.
5. Добавлен переключатель положения гидрографии:
   - поверх административного слоя;
   - под административным слоем.
6. Слой озёр фильтруется по времени:
   - до 1959 года скрываются объекты, распознанные как водохранилища;
   - с 1959 года водохранилища показываются.
7. Озёра/водохранилища рисуются выше рек внутри гидрографического блока, чтобы линии рек не проходили поверх акваторий.
8. Интерфейс адаптирован под разные экраны: на широком мониторе — две боковые панели и карта между ними; на узких экранах — последовательная вертикальная раскладка.

Для публикации на GitHub Pages загрузить в репозиторий содержимое этой папки, не ZIP-файл.
