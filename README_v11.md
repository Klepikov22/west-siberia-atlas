# Western Siberia Web MVP v11

Изменения v11:

- Режим «Рука / перемещение» оставлен как базовый курсор: карта двигается, одиночный клик по району/уезду добавляет или убирает объект из выборки.
- Прямоугольная и полигональная выборка теперь работают по пересечению геометрии, а не по центроиду. Объект попадает в выборку, если рамка/полигон касается его геометрии.
- Дополнено объединение северных фрагментов после океанической обрезки:
  - 1926: Обдорский;
  - с 1979: объекты с «Надым» в названии;
  - сохранены правила для Берёзовского, Тазовского, Ямальского и Приуральского.
- Площадь и плотность пересчитаны после dissolve.

Открывать после публикации: `?v=11`.
