# v141 — синяя защита ВШЭ, белый фон, население 1855/1876

Что сделано:

- Добавлен отдельный стиль `hseDefenseBlue` / «ВШЭ · защита ВКР · синяя» для районов и подложки.
- В синем режиме фон карты принудительно белый, без декоративных градиентов.
- Гидрография в синем режиме усилена: голубые линии/заливки остаются читаемыми на белом фоне.
- Добавлен отдельный отключаемый слой «Контур АТД-1 тёмно-синим».
- При первом открытии включены только: гидрография, административный слой, контур АТД-1 и железные дороги. Центры, подписи точек, круги населения, топологические/специальные слои по умолчанию выключены.
- Население перенесено в слои 1855 и 1876 гг. из переданных пользователем файлов по `unit_id`/названию.
- Геометрия, `area_km2`, топология, ЖД-поля и смежность в слоях 1855/1876 сохранены из v140, чтобы не откатить расчётные площади.
- Подписи центров/городов при включении чекбокса теперь строятся как интерактивные Leaflet-div-метки: клик по подписи выводит карточку центра в правой панели.

Диагностика населения:

- 1855: 26 объектов, население заполнено у всех, сумма по проектному слою — 2 234 432.
- 1876: 27 объектов в проектном слое, население заполнено у всех этих 27 объектов, сумма — 3 291 163.
- В пользовательском файле 1876 было 29 объектов и сумма 3 291 273; два дополнительных безымянных объекта отсутствуют в проектном v140-слое, поэтому геометрически не импортировались.
- Сверка с соседями выполнена на уровне сопоставимых верхних АТЕ: Тобольская и Томская линии 1848→1855→1876→1897 монотонны; Семипалатинская и Акмолинская/Область Сибирских Киргизов сопоставлялись прежде всего в ряду 1855→1876→1897.

Техническая диагностика лежит в `docs/v141_population_and_style_diagnostics.json`.
