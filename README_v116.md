# v116

Исправление интерфейса после v115:

- добавлен видимый переключатель «Подписи городов / точек» рядом со слоем городов/центров;
- подписи городских/центровых точек строятся отдельным Leaflet/HTML-слоем и показывают название и компактное население;
- добавлена кнопка «Таблицы метрик по годам» рядом с динамикой метрик;
- таблицы сгруппированы по компонентам анализа: АТЕ и площади, население, ЖД, соседство, граф и топология, разброс площадей;
- в таблицах можно включать/выключать годы и метрики, менять ориентацию, копировать TSV и скачивать CSV.
