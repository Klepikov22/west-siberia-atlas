# Western Siberia Atlas MVP v3

Изменения v3:

- добавлен слой океана `data/ocean/ocean.geojson`;
- административные полигональные слои 1897–2021 обрезаны по маске океана, то есть из них вычтён океанический полигон;
- пересчитаны `area_km2` и `density` после обрезки;
- добавлена выборка объектов на карте: клик по району/уезду переключает объект в выбранные;
- статистика по умолчанию считается по всему показанному административному слою, а при выборе одного или нескольких объектов — по выборке;
- добавлены кнопки очистки выборки и выбора всех объектов текущего года;
- сохранены светлая/тёмная тема, плавный зум и мягкая рамка экстента из v2.

Файл `docs/v3_ocean_clip_summary.csv` показывает, сколько площади было удалено по океану для каждого года.
