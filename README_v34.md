Версия v34

Что исправлено и добавлено:
- экспортная карта теперь строится как отдельная live-vector Leaflet-карта внутри макета, а не как неудачный скриншот основного div;
- добавлены форматы A4 horizontal, A4 vertical и широкий экран;
- добавлены режимы оформления: карта для диплома, карта для статьи, презентационный слайд;
- добавлена автоматическая академичная подпись «Составлено автором по материалам…»;
- столбцы населения убраны, оставлены только круги населения;
- к выбору нормирования добавлена справка при наведении: линейное, квадратный корень, логарифмическое, квантильное.
