# Версия v86

## Расчёт соседства административных объектов

Для всех административных слоёв рассчитано геометрическое соседство АТЕ.

### Метод

- Соседство считается по общей границе полигонов.
- Порог общей границы: не менее 0,5 км.
- Контакт только в точке соседством не считается.

### Исключения из расчёта

Из графа соседства исключены:

- спорные территории;
- территории с неясным или слабым статусом контроля;
- двоеданческие волости и сходные специальные зоны;
- контекстные территории, исключённые из аналитики;
- города / горсоветы площадью менее 50 км².

Для исключённых объектов заполнены:

- `adjacency_excluded = true`;
- `adjacency_exclusion_reason`.

### Новые поля

Для каждого объекта добавлены поля:

- `adjacent_unit_ids` — список `unit_id` соседних объектов;
- `adjacent_names` — список названий соседних объектов;
- `adjacent_count` — число соседей;
- `adjacent_units` — строка с названиями соседей для просмотра в карточке;
- `adjacency_method` — описание метода расчёта;
- `adjacency_excluded` — исключён ли объект из расчёта;
- `adjacency_exclusion_reason` — причина исключения.

### Диагностика

В `docs/` добавлены:

- `v86_adjacency_edges.csv` — таблица пар соседства;
- `v86_adjacency_excluded_features.csv` — исключённые объекты;
- `v86_adjacency_summary.csv` — сводка по годам.
