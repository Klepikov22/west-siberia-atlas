# v77

## Изменения интерфейса

- Таймслайдер возвращён к фиксированной ширине уровня до добавления дополнительных срезов 1947 и 1964 годов.
- Временная шкала переведена в режим карусели:
  - все годы остаются доступны;
  - активный год центрируется внутри внутренней дорожки, а не через `scrollIntoView` страницы;
  - добавлены кнопки перехода к предыдущему/следующему году;
  - выбор поздних годов больше не должен сдвигать основной интерфейс влево.
- Убрано page-level прокручивание при активации поздних дат: прокрутка происходит только внутри `#yearTimeline`.

## Проверка дублей 1897, 1914, 1918, 1923

- 1897: дублей по `unit_id`, `name + admin_parent + unit_type`, `name + admin_parent` не найдено.
- 1914: дублей по `unit_id`, `name + admin_parent + unit_type`, `name + admin_parent` не найдено.
- 1918: чистых дублей `unit_id` не найдено; остались 3 одноимённые исторические/спорные группы, не объединялись автоматически.
- 1923: чистых дублей `unit_id` не найдено; осталась 1 нулевая спорная территория, не объединялась автоматически.
