# v127 — графики приуроченности границ

Добавлено модальное окно «Графики факторов границ».

- Мультикомпонентный столбчатый график по годам для природной морфологии границ.
- Геометризированные участки вынесены как подтип природно необъяснённых границ.
- Отдельный столбчатый график по годам для приуроченности к железным дорогам.
- Интерактивная легенда: факторы можно отключать; после отключения столбцы перестают быть нормированными до 100% и показывают остаточную сумму на шкале 0–100%.
- Подсказки при наведении показывают год, фактор, процент, километры, число АТЕ и суммарный периметр.
- Графики пересчитываются при смене выборки, верхнеуровневых фильтров и числовых фильтров.
