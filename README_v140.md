# v140 — стиль ВШЭ для карт защиты

Добавлен отдельный картографический стиль для подготовки чистых карт к презентации ВКР.

## Что изменено
- Новый пункт в «Стиль районов»: **ВШЭ · защита ВКР**.
- Новый пункт в «Стиль подложки»: **ВШЭ · защита ВКР**.
- При выборе стиля районов ВШЭ подложка автоматически переключается в такой же режим.
- Цвета АТЕ верхнего / среднего / нижнего уровня переводятся в спокойную HSE-палитру.
- Для основных исторических и современных регионов задано устойчивое соответствие цветов: Тобольская/Тюменская — светло-синий, Томская — синий, Омская — тёмно-синий, Новосибирская/Кемеровская/Алтайский край — красная шкала и т.п.
- Пунсоны городов становятся красными с белой обводкой.
- Подписи городов и центров получают белую плашку, синий текст и красный акцент для крупных городов.
- Гидрография становится максимально лёгкой, почти фоновой.
- Железные дороги становятся тоньше и серее, чтобы не превращать карту в «паутину».
- Экспортные SVG/PNG также подхватывают стиль для АТЕ, кругов населения и железных дорог.

## Как использовать
1. Открыть `index.html`.
2. В панели слева выбрать:
   - **Стиль районов → ВШЭ · защита ВКР**;
   - подложка переключится автоматически, но можно проверить **Стиль подложки → ВШЭ · защита ВКР**.
3. Выбрать нужный год, режим отображения и включить/выключить подписи, центры, ЖД и население.
4. Экспортировать PNG через штатный режим экспорта или сделать скриншот.
