# v95

Исправление слоя устойчивости границ.

Главное отличие от v94: устойчивость теперь выводится по реальным фрагментам границ нормальных АТЕ, а не только по точным общим внутренним пересечениям полигонов и не по графовым/центроидным перемычкам. В расчёт включены внутренние границы и внешние / экспонированные контуры нормальных АТЕ, поэтому в слое появляются речные и орографические рубежи, включая участки по Иртышу, Уралу и внешнему контуру Горного Алтая, если они повторяются в исторических срезах.

Метод прототипа v95:
- период расчёта: 1700–1923;
- опорные годы для отображаемых представителей: 1780, 1805, 1897, 1914, 1923;
- сравниваются реальные линейные фрагменты границ;
- допуск: 15 км севернее / на 59° с. ш., 10 км южнее;
- спорные, неясные, слабоконтрольные, двоеданческие и передаваемые территории исключены как источники границ;
- результат: `data/stability/boundary_stability_v95.geojson`.

Ограничение: это обзорный прототип для читаемой веб-карты. Для публикационной версии стоит заменить быстрый midpoint+direction match на более тяжёлую линейную кластеризацию коридоров, но геометрия вывода уже является настоящей границей, а не K-line / centroid connector.
