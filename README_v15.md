# Western Siberia historical atlas — v15

Изменения:

- Убраны постоянные подписи центров и соответствующая галочка.
- Увеличены элементы левой панели: шрифт, чекбоксы, область клика.
- Центры сделаны кликабельными: при клике подсвечиваются и показывают карточку в правой панели.
- Инструмент «Рука» переименован в «Курсор»; он работает для одиночного выбора.
- Если выбран один объект, его карточка показывается сразу; выпадающий список появляется только при множественной выборке.
- Добавлены расчетные поля по ЖД: `rail_length_km`, `rail_density_km_1000`, `rail_segments_count`.
- В правой панели добавлены диаграммы по верхнеуровневым единицам: средняя площадь, среднее население, средняя плотность, средняя плотность ЖД. Города/малые полигоны исключаются фильтром `area_km2 >= 700`.
