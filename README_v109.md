# v109

## Основные изменения

1. Добавлена опорная реконструкция населения для ранних слоёв: 1783, 1809, 1821, 1838 и 1848 гг.
2. В административные GeoJSON добавлены унифицированные поля источниковой верификации:
   - `population_source_year`
   - `population_source_unit`
   - `population_basis`
   - `population_quality`
   - `population_method`
   - `population_note`
   - `population_source`
   - `population_reconstruction_version`
3. В `population_long.csv` добавлены ранние годы и те же поля источниковой верификации.
4. Для 1848 г. учтён хвост Семипалатинского и Усть-Каменогорского уездов как отдельная оценочная добавка к томской рамке, с сохранением топологического исключения для степного/контекстного полигона.
5. Слой 1897 г. заменён пользовательским `admin_1897.geojson`: Семипалатинский и Усть-Каменогорский уезды разделены, Усть-Каменогорскому присвоен уникальный `unit_id = adm_1897_31`.
6. Для 1897 г. пересчитаны площади, топология, узлы, рёбра, графовые метрики, ЖД-показатели и строка в `multiyear_metrics_by_year.json`; население сохранено по присланному слою.

## Диагностика

См. файлы `docs/v109_*`:
- распределение раннего населения по АТЕ;
- контрольные суммы по источниковым рамкам;
- сравнение метрик до/после;
- проверка замены слоя 1897;
- ребра топологического графа 1897.
