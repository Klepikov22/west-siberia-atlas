# v131

## Главное
- Убран отдельный мини-блок «Продвинутый граф · коридоры и барьеры».
- Продвинутый граф теперь управляется так же, как обычный топологический граф:
  - `Рёбра потенциальной связности`;
  - `Узлы продвинутого графа / JSON-точки`.
- Режимы продвинутого графа доступны в общем селекторе режима карты и окрашивают полигоны АТЕ независимо от включения рёбер/узлов.
- Пересчитаны показатели длины и плотности железных дорог: вместо длины в Web Mercator используется геодезическая длина после клиппинга в локальной LAEA-проекции.

## Почему пересчитаны ЖД
Предыдущие значения были завышены на широтах Западной Сибири из-за использования метрической длины в Web Mercator. Для 1989 сумма по АТЕ уменьшилась примерно с 16,18 тыс. км до 8,68 тыс. км, что ближе к контрольной оценке пользователя.
