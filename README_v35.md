Версия v35

Что изменено:
- постоянные подписи на карте теперь строятся по полигонам административных единиц, а не по точкам центров;
- подпись берётся из атрибутов самого административного полигона: name / unit_name / admin_name / unit_id;
- подписи центров отключены, чтобы не смешивать города/центры с подписями районов, уездов и округов;
- генерализация подписей выполняется по населению: при конфликте и недостатке места сохраняется подпись более населённой АТЕ;
- добавлены разные уровни подписи: крупные, средние и малые АТЕ;
- экспортная карта также получает подписи административных полигонов, а не точек центров.
