# Western Siberia Web Atlas — v17

Изменения:
- Добавлена иерархия подписей центров.
- Города определяются по признакам `г.`, `город`, `горсовет`, а для ранних срезов — по наличию городского населения `Pop_Urban` у точечного центра без административного суффикса.
- Крупные города подписываются жирнее: >20 тыс. до 1926 г.; >50 тыс. в 1926–1939 гг.; >100 тыс. после 1939 г.
- Hover-панель усилена: появляется через 0,5 сек, следует за курсором, не вылезает за край экрана.
