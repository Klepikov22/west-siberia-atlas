# Western Siberia Historical Atlas — v8

Стабилизирующая версия после проблем с порядком слоёв.

## Что изменено

- Упрощён порядок слоёв: без custom panes и без Canvas-рендеринга.
- Слои каждый раз пересобираются в стабильном порядке:
  1. реки;
  2. океан + озёра + водохранилища;
  3. административные полигоны;
  4. железные дороги;
  5. круги населения;
  6. центры;
  7. подписи.
- Административный слой добавлен отдельной галочкой.
- Рельеф отсутствует полностью.
- Минимальный зум: 3.5.
- Добавлены инструменты:
  - рука / перемещение;
  - выбор прямоугольником;
  - выбор полигоном.
- До 1959 года водохранилища скрываются из водного слоя.
