# v66

Исправления по интерфейсу Full HD и экспортной компоновке.

## Интерфейс карты
- Панель «Верхнеуровневые АТЕ» в свернутом виде теперь держит заголовок и кнопки в одну строку.
- Кнопки в свернутой панели выровнены по высоте, с одинаковыми внутренними отступами и равными промежутками.
- Для Full HD / 1080p добавлена правка положения стрелок сворачивания боковых панелей:
  - при открытых панелях стрелки стоят у фактической кромки панели;
  - при скрытых панелях стрелки уходят к краю окна.
- Для плавающих панелей фильтров и верхнеуровневых АТЕ снят избыточный нижний «барьер»: при перетаскивании нижний допуск теперь считается по нижнему отступу таймслайдера.

## Экспорт карты
- Экспортная карта в Ламберте возвращена в нормальную ориентацию «север сверху».
- Авторазмещение внутренней картографической рамки теперь считает аспект по проецированному экстенту выбранных полигонов, а не по грубой широте/долготе.
- Базовая подгонка слоя к внутренней рамке строится по реальным координатам полигонов в Ламберте.
- По умолчанию минимальный экранный зазор от границы слоя до внутренней рамки — 10 px.
- Поля «Север / Восток / Юг / Запад» в километрах работают как дополнительный буфер к этим 10 px, без повторного двойного раздувания экстента.

## Технически
- Версия обновлена до `APP_VERSION = 66`.
- Подключения обновлены до `app.js?v=66` и `style.css?v=66`.
- `app.js` проходит `node --check`.
