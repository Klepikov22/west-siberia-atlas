# v74 — очистка дублей и стабилизация интерфейса

Изменения:

- Убраны дубли в слоях 1959, 1947 и 1964.
- 1959: объединены 4 повтора unit_id без двойного счёта населения.
- 1947: объединены повторяющиеся фрагменты АТЕ, включая Колпашевский горсовет / район; городская и сельская компоненты разведены по `urban_pop` и `rural_pop`.
- 1964: объединены повторяющиеся фрагменты АТЕ с суммированием статистики.
- Для 1947 выполнена мягкая перебалансировка городского/сельского населения по структуре 1959 с макроконтролем 43,5% городского населения для контура без Тюмени, ХМАО и ЯНАО; общая численность АТЕ сохранена.
- Добавлены служебные поля сопоставимых контуров: `project_ws_full`, `classic_ws_laperdin`, `classic_ws_burmatov`.
- Таймслайдер больше не вызывает `scrollIntoView` по документу: поздние годы прокручиваются только внутри дорожки таймлайна.
- Для Full HD отключён CSS `zoom` у перетаскиваемых панелей фильтров и верхнего уровня, чтобы убрать рывки и съезд при drag.
