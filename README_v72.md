# v72

Изменения:

- Подключены административные срезы 1848, 1855 и 1876 гг.
- Новые слои приведены к общей схеме `admin_*`: `unit_id`, `year`, `name`, `unit_type`, `admin_parent`, `admin_intermediate`, `admin_superparent`, `area_km2`, `density`, `rail_length_km`, `rail_density_km_1000` и служебные поля источника.
- Слои добавлены в `data/manifest.json` и на таймслайдер.
- Добавлен переключатель периода таймлайна: все годы, XIX век, XX век, XXI век, имперский период, советский период.
- Добавлена кнопка слайд-шоу для последовательного просмотра изменения сетки АТД в выбранном периоде.
- Сохранено изменение v71: минимальная ширина внешней PNG-рамки экспорта снижена до 500 px.

Ограничение данных: в исходных слоях 1848/1855/1876 не задано население, поэтому демографические режимы для этих годов будут работать ограниченно; корректно доступны геометрия, площади, верхнеуровневая принадлежность, типы единиц и объектная статистика.
