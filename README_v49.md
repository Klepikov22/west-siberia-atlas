# Версия 49

Исправления экспорта карты:

- пересобран архив заново, чтобы не было проблемы с выгрузкой v48;
- добавлена автоматическая подгонка картографического поля под форму выбранного слоя / выборки;
- вертикальные выборки получают более узкое поле, горизонтальные — более низкое поле;
- картографическое поле рассчитывается по административным полигонам выбранного охвата;
- добавлено ограничение компоновки: верхний и нижний интервалы от поля карты до внешней рамки не меньше 100 px;
- боковые интервалы от поля карты до внешней рамки автоматически не превышают 400 px;
- в панели экспорта добавлен переключатель автоподгонки и строка статуса с текущими размерами поля;
- версия проекта обновлена до v49.
