# v139

Версия переводит продвинутый граф в более явную стоимостную трактовку: ребро смежности имеет условную стоимость перехода, которая снижается транспортными/речными коридорами и повышается природными барьерами.

Изменения:
- уточнено правило учёта рельефа >400 м: западнее 84°E — севернее 50°N, восточнее 84°E — севернее 49°N;
- всё течение Иртыша добавлено в судоходную речную компоненту;
- пересчитаны `adv_*` поля рёбер, узлов и административных слоёв;
- фильтр рёбер получил числовой ввод минимальной силы и максимального импеданса.
