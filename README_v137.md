# v137 — компонентная модель продвинутого графа связности

В этой версии продвинутый граф потенциальной связности переведён с взаимоисключающих типов рёбер на компонентную модель.

## Что изменено

- Каждое ребро смежности получает отдельные положительные компоненты:
  - `adv_rail_score` — железнодорожная компонента;
  - `adv_old_road_score` — трактовая / историко-дорожная компонента;
  - `adv_navigable_river_score` — судоходная речная компонента.
- Отдельно считаются компоненты импеданса:
  - `adv_wetland_impedance`;
  - `adv_relief_impedance`;
  - `adv_watershed_impedance`;
  - `adv_minor_river_impedance`.
- Итоговые поля:
  - `adv_positive_corridor_score`;
  - `adv_positive_bonus`;
  - `adv_barrier_impedance_multiplier`;
  - `adv_impedance`;
  - `adv_strength`;
  - `adv_edge_class` / `adv_edge_label`.
- В легенде рёбра теперь трактуются как мультикомпонентные классы: коридор без существенного импеданса, коридор + природный импеданс, обычное соседство + тип барьера и т. д.
- Добавлены режимы карты для отдельных компонент продвинутого графа: ЖД, тракт/дороги, судоходные реки, болотный, рельефный и водораздельный импеданс, средняя сила и средний импеданс рёбер.

## Интерпретация

Модель показывает не фактическую историческую транспортную доступность, а эвристическую потенциальную связность административных единиц: транспортные и речные коридоры усиливают связь, природные барьеры повышают импеданс и ослабляют её.
