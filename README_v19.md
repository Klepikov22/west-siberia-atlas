# v19 — исправление подписей через Leaflet tooltip

Подписи центров реализованы как permanent tooltip, привязанные к точечным маркерам Leaflet.
Это должно решить проблему, когда у части пользователей подписи оказывались привязаны к экрану, а не к карте.

Остальная функциональность сохранена от v17: иерархия городов, hover-панель, Pop_urban для центров, правая аналитика.
