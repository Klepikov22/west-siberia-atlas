# v96

Правки слоя устойчивости границ:

- период расчёта расширен до полного ряда `1700–2021`;
- источник слоя — реальные фрагменты контуров нормальных АТЕ, без K-линий и центроидных перемычек;
- в v95 искусственные разрывы давало прореживание представителей: соседние участки удалялись, если их середины были ближе 45 км; в v96 это прореживание убрано;
- на карту добавлен выбор периода `от / до`, фильтрация пересчитывает число совпавших срезов в выбранном периоде по списку `matched_years`;
- минимум срезов по умолчанию снижен до `4+`, чтобы обзорный слой не выглядел как оборванная пунктирная сетка;
- допуск сохранён: 15 км севернее/на 59° с. ш., 10 км южнее 59° с. ш.;
- исключения сохранены: спорные, неясные, слабоконтрольные, двоеданческие, передаваемые территории, а также малые городские контуры.

Файлы данных:

- `data/stability/boundary_stability_v96.geojson`
- `data/stability/boundary_stability_v96_summary.json`

Оговорка: это по-прежнему обзорный слой геометрической устойчивости. Для финальной публикационной версии лучше затем агрегировать повторяющиеся linelets в единые коридоры, чтобы убрать дублирующиеся линии разных опорных лет.
