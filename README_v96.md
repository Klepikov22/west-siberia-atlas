# v96 — узлы графа как обычный слой карты

- Добавлен рядовой чекбокс в общей панели слоёв: `Узлы графа / центроиды АТЕ`.
- Узлы больше не зависят от старого рендера топологического графа: они строятся заново по центрам текущих административных полигонов, тем же общим способом, что и пользовательские оверлеи карты.
- Добавлен отдельный рядовой чекбокс `Рёбра смежности`; рёбра и узлы теперь включаются из общей панели элементов карты.
- Центроиды фильтруются по текущему слою, метрикам, родительским фильтрам и выборке; исключаются спорные/слабоконтрольные/служебные объекты и полигоны меньше 50 км².
- Узлы кликабельны: выборка и карточка объекта работают по клику на центроид.
- Версия интерфейса обновлена до v96.
